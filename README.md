# Bullet Hell Sandbox

A small bullet hell sandbox made with C++ and Raylib. You can add multiple spawners, change bullet colors before spawning, change bullet velocity etc.

## Features

Multiple bullet spawners
Adjustable velocity, count, interval and bullet color
Sound effects toggle


## Controls
V - Toggle spawn mode
Y - Toggle UI
Left Click (while in spawn mode) - Place spawner

## Build Project
To build the project, open a terminal in the directory containing premake5.lua
Then run:

<i> Tools\premake\premake5.exe vs[generator] </i>


for example:

<i> Tools\premake\premake5.exe vs2026 </i>

<i> Tools\premake\premake5.exe gmake2 </i>
