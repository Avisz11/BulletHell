# Bullet Hell Sandbox

A small bullet hell sandbox made with C++ and Raylib. You can add multiple spawners, change bullet colors before spawning, change bullet velocity etc.

## Features

Multiple bullet spawners
Adjustable velocity, count, interval and bullet color
Sound effects toggle


## Controls
V - Toggle spawn mode
Y - Toggle UI
Left Click (while in spawn mode) - Place spawner

## Build Project
To build the project, open a terminal in the premake5 directory.
Then run:

<i> premake5 <generator> </i>
for example:
<i> premake5 vs2026 </i> 
<i> premake5 gmake2 </i>
